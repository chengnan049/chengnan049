来自微软的大宝贝 Windows Terminal

发表于9月-19-21|更新于10月-1-21
|字数总计:1.1k|阅读时长:4分钟|阅读量:

前言

万物起于Terminal，那也就让Terminal为本博客做开篇吧。我是一个重度终端用户，每天都要用到终端，因此拥有一个好看且顺手的终端自然是必不可少的。在写这篇文章之前我也翻阅了不少文档及博客，但他们都有一个共病，那就是一旦这些插件更换了地址代码就自然会失效，就固然给新手带来了许多麻烦，而且也不标注出处（此处请容许我暴打这些作者），因此还是放在自己这里好，也方便下次使用。

选用

作为一个软粉自然要使用自家的软件啦，这里选用PowerShell 7 + Oh My Posh作为终端环境，微软在Build 2019开发者大会上公布了自家最新的Windows Terminal并宣布开源该项目，此外支持WSL及其他终端。从此Cmd与传统PowerShell的时代也逐渐走向尽头，目前该项目在GitHub开源，并提供多种安装方式。

安装Windows Terminal

通过 Microsoft Store [推荐]

Windows Terminal from the Microsoft Store

通过 GitHub

从Releases page下载Microsoft.WindowsTerminal_<versionNumber>.msixbundle （使用此方式安装将不会自动更新）

通过 WPM

通过 Scoop（非官方）

P.S.通过此方式安装的Windows Terminal将无法设置为默认终端,如遇到任何安装/更新问题请在Issues page提交Issues

通过 Chocolatey（非官方）

安装新版 PowerShell

参考：微软PowerShell安装文档

Windows 10自带的PowerShell为PowerShell 5，两者为共存关系，无法覆盖

通过 Microsoft Store [推荐]

Stable版本 Preview版本

通过 GitHub

Stable版本 Preview版本

PowerShell-<version>-win-x64.msi，无脑Next即可

  • 默认安装目录为$env:ProgramFiles\PowerShell\<version>

  • 启动目录为$env:ProgramFiles\PowerShell\<version>\pwsh.exe

通过WPM安装

通过 Scoop（非官方）

P.S.使用此方式安装的 PowerShell 需手动添加至 Windows Terminal

抱歉，我使用Scoop，请Chocolatey用户自行摸索

安装Oh My Posh

参考：Oh My Posh Docs

打开Windows Terminal，如果你从Microsoft Store或者GitHub安装PowerShell，默认终端已经配置为PowerShell 7

macOS及Linux用户一定也知道Oh my zsh项目，Oh My Posh也是一个优秀的终端优化的项目

执行以下命令安装Oh My Posh

Install-Module oh-my-posh -Scope CurrentUser

输入y回车即可（从不被信任的仓库安装）

配置Oh My Posh

使用Notepad对PowerShell默认启动脚本进行配置，提示不存在点击“是创建Microsoft.PowerShell_profile.ps1文件，目录在C:\Users\Username\Documents\PowerShell

输入以下内容并保存

更多主题执行Get-PoshThemes或访问Themes

更改主题后可能会出现字体乱码的情况，我们可以安装PL字体进行解决，主流的字体有Nerd Fonts,Cascadia Code等等，我这里使用[Cascadia Code](microsoft/cascadia-code: This is a fun, new monospaced font that includes programming ligatures and is designed to enhance the modern look and feel of the Windows Terminal. (github.com))进行演示

安装Cascadia Code

通过 GitHub

在Releases页面找到CascadiaCode-<version>.zip，下载后解压ttf文件夹内所有文件右键Install for all users

通过 Scoop

美化及调整操作习惯

点击下箭头Settings > 将 Default terminal application 更改为刚刚安装的 Windows Terminal 并点击保存（以后不再赘述）

打开 Show acrylic in tab row (requires relauch)

切换Color schemes为One Half Light，并将Font face更换为Cascadia Mono PL，将Font Size调整为12，Font weight调整为Semi-light打开Arcylic选项，将Arcylic opacity调整为50%

隐藏旧版本PowerShell及Azure Cloud Shell

至此关于 Windows Terminal 的调教基本完成，现在我们来配置VS Code

调整 VS Code 字体

按下Ctrl + Shift + P 输入Open Settings，点击第一个

“editor.fontWeight”: 350,

“editor.fontSize”: 15.5,

“editor.fontFamily”: “Cascadia Mono”,

将上述内容粘贴保存即可
